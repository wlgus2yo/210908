html 4.01 , xhtml 에서는 iframe을 사용하고, html5 일 경우 embed 사용 권장

  • autoplay : video, audio 동일, 자동 재생(muted 필수)
  • controls : video, audio 동일, 재생영역에 제어버튼 활성화
  • loop : video, audio 동일, 반복 재생 활성화
  • muted : 음소거 활성화
  • src : 경로 설정
  • width, height : 너비, 높이
  • poster : video 전용 속성, video를 불러오지 못할 때, 대체 이미지 출력
